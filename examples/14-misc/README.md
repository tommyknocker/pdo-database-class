# Miscellaneous Examples

Various examples and demonstrations extracted from the main README.md.

## Examples

### 01-readme-examples.php
Examples from the main README.md demonstrating key features.

**Topics covered:**
- Basic CRUD operations
- Filtering and joining
- JSON operations
- Transactions
- Raw queries
- Complex conditions
- Helper functions
- Error handling
- Query analysis
- Connection management
- Batch processing
- Bulk operations
- Enhanced error diagnostics

**Supported databases:**
- MySQL
- MariaDB
- PostgreSQL
- SQLite
- Microsoft SQL Server (MSSQL)

## Usage

```bash
# SQLite (default)
php 01-readme-examples.php

# MySQL
PDODB_DRIVER=mysql php 01-readme-examples.php

# MariaDB
PDODB_DRIVER=mariadb php 01-readme-examples.php

# PostgreSQL
PDODB_DRIVER=pgsql php 01-readme-examples.php

# Microsoft SQL Server
PDODB_DRIVER=sqlsrv php 01-readme-examples.php
```
